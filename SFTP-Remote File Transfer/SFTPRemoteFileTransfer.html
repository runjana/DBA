SFTP

Secure File Transfer Protocol (SFTP) is a secure version of File Transfer Protocol (FTP), which facilitates data access and data transfer over a Secure Shell (SSH) data stream. It is part of the SSH Protocol. This term is also known as SSH File Transfer Protocol.

                                        Fig: SSH

Steps:

  1. Install Openssh-server (On linux computer).
  2. Install ssh client (WinSCP)
  3. Write the bash script to connect and transfer the file to remote pc and run it

1.  Install Openssh-server:

In our task, we installed openssh server in linux machine

                                Fig 1.1: Install openssh server

Then, we need to start ssh service. We can do this using

                        $sudo service ssh start

Now, this linux machine is a ssh server and clients can connect to it.

If the client is also in the linux machine, then the client can connect using a simple single line command i.e.

$ssh remote_pc_username@hostname_or_ip_address

        Fig 1.2: Connect to remote pc

At last, we have to close the ssh service using command

$ssh service stop

2. Install ssh client:

There are many ssh clients like Putty, Bitvise SSH, XShell 4, MobaXterm, WinSCP in internet  in our case, we used WinSCP.

We downloaded and installed WinSCP from its official site.
